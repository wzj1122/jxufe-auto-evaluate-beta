# 江西财经大学自动评教 (Beta)

适用于江西财经大学 KINGOSOFT 教务系统的自动评教油猴脚本（测试版本）。

## 功能

- 浮动控制面板（可拖拽）
- 快速模式：连续评教不刷新
- 兼容模式：每次评教后刷新页面（v1.0.0 逻辑）
- 自动切换：快速模式失败时自动切换兼容模式
- 清除评教：一键删除暂存数据（面板按钮）
- 会话过期检测与自动恢复
- 实时进度显示

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 Tampermonkey 图标 → 添加新脚本
3. 将 `auto-evaluate.user.js` 的内容粘贴进去
4. 保存（Ctrl+S）
5. 打开 `https://jwxt.jxufe.edu.cn/frame/homes.action*`

## 使用

### 浮动面板（页面右下角）

| 按钮 | 功能 |
|------|------|
| 开始 | 开始自动评教 |
| 暂停 | 暂停/继续评教 |
| 停止 | 停止评教 |
| 快速 | 连续评教不刷新（默认） |
| 兼容 | 每次评教后刷新页面 |
| 清除评教 | 逐个删除暂存数据 |

### 操作流程

1. 打开教务系统，页面右下角出现控制面板
2. **清除评教数据**：先导航到评教页面 → 点击「清除评教」→ 脚本自动逐个删除
3. **开始评教**：点击「开始」→ 选择模式 → 等待完成
